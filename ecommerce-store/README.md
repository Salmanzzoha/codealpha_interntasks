# CodeAlpha Internship — Full Stack Development Tasks

Completed project tasks for the CodeAlpha Full Stack Development internship (July–August 2026).

📁 Projects

1. [task-manager](./task-manager) — Task Manager (To-Do App with Auth)
A full-stack task manager with user authentication. Users register/log in and manage a private list of tasks.
- **Stack:** Node.js, Express, SQLite, JWT auth, bcrypt — vanilla HTML/CSS/JS frontend
- **Features:** register/login, per-user task isolation, create/edit/complete/delete tasks

2. [feedback-board](./feedback-board) — Feedback / Review Board
A public feedback board where anyone can submit a star-rated review and browse/sort submissions. No login required.
- **Stack:** Node.js, Express, SQLite — vanilla HTML/CSS/JS frontend
- **Features:** submit feedback (name, message, 1–5 star rating), sort by newest/highest rated/most upvoted, upvote button

 Running either project
Each folder has its own README with full setup steps. In short:
```bash
cd <project-folder>/backend
npm install
cp .env.example .env
npm start
```
Then open `<project-folder>/frontend/index.html` (e.g. via VS Code's Live Server extension) to use the app.
